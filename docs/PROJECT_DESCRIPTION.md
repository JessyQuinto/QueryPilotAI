# Descripción del Proyecto

**InsightForge AI** es un agente de ingeniería analítica orientado a fraude que traduce preguntas de negocio en consultas SQL validadas, seguras y auditables. El sistema descompone la intención analítica, aplica guardrails de seguridad, genera SQL robusto sobre vistas certificadas, ejecuta la consulta en Azure SQL y entrega insights ejecutivos con transparencia operativa.

## Challenge resuelto
Los usuarios empresariales necesitan respuestas fiables a partir de los datos, pero la generación manual de SQL y la dependencia de especialistas crean demoras, errores y riesgos de seguridad. InsightForge AI reduce ese cuello de botella al combinar IA generativa, validación de políticas, aprobación humana y observabilidad.

## Diferenciales
- Transparencia del razonamiento operativo.
- SQL gobernado con controles de riesgo.
- Flujo de aprobación para consultas sensibles.
- Observabilidad y auditoría de punta a punta.
- Enfoque verticalizado en fraude.
